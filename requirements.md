# Функциональные требования (выдержка)

## Корзина

*   REQ-01: Пользователь может добавить товар в корзину из карточки товара
*   REQ-02: Пользователь может изменить количество товара в корзине; итоговая сумма пересчитывается мгновенно
*   REQ-03: При установке количества 0 товар удаляется из корзины
*   REQ-04: Пользователь может удалить отдельный товар
*   REQ-05: Пользователь может полностью очистить корзину
*   REQ-06: Промокод применяется один раз; скидка суммируется с акционной ценой товара
*   REQ-07: Кнопка «Оформить заказ» неактивна, если корзина пуста

## Оформление заказа

*   REQ-08: Неавторизованный пользователь перенаправляется на страницу входа
*   REQ-09: Авторизованный пользователь видит форму подтверждения адреса и выбора оплаты
*   REQ-10: Поддерживаются способы оплаты: карта, электронные деньги, наличные при доставке
