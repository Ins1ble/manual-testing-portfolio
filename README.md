# Тестовое портфолио

Привет, меня зовут Анатолий, и я целенаправленно готовлюсь к стажировке в Т1 по направлению «Ручное функциональное тестирование».  
Этот репозиторий — живой пример моей тестовой документации, созданный, чтобы показать реальные навыки, которые перечислены в требованиях к кандидатам.

Я не просто прошёл теорию, а применил её на практике: от анализа требований до проверки API и работы с базами данных.

## Какие навыки здесь демонстрируются

*   Теория тестирования: чек-листы, тест-кейсы, баг-репорты
*   Техники тест-дизайна: классы эквивалентности, граничные значения, попарное тестирование
*   Модели разработки ПО: Agile и Waterfall (учтены в тест-плане)
*   Работа в командной строке Linux: чтение логов, фильтрация, поиск ошибок
*   Анализ логов: консоль браузера, серверные логи
*   Понимание REST и SOAP: проверка API через Postman, тест-кейсы для REST
*   SQL: написание запросов для валидации данных в БД
*   Понимание TCP/IP: базовое понимание стека при анализе сетевых запросов

## Структура проекта

*   `test_plan.md` — тест-план
*   `requirements.md` — требования
*   `checklist.md` — чек-лист
*   `bug_reports.md` — баг-репорты
*   `test_cases.md` — тест-кейсы (UI и API)
*   `traceability_matrix.md` — матрица трассируемости
*   `sql_queries.md` — SQL запросы
*   `linux_commands.md` — работа в Linux (логи)
*   `postman_collection.json` — коллекция Postman

Всё это сделано мной самостоятельно, живым языком, без шаблонов.  
Я готов учиться дальше и применять эти навыки в реальных задачах Т1.
